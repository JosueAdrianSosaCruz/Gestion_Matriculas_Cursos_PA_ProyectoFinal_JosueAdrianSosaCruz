Cómo usar el programa
Al ejecutar el programa se muestra un menú principal, desde el cual se accede a todas las funcionalidades del sistema.
Menú principal
Opciones disponibles:
Crear curso
Mostrar cursos
Gestión de curso
Gestión de estudiantes
Gestión de notas
Reportes
Salir
Crear curso
Seleccione la opción 1
Ingrese el nombre del curso
El curso quedará activo y disponible para su gestión
El sistema permite un máximo de 10 cursos.
Mostrar cursos
Seleccione la opción 2
Se mostrará la lista de cursos creados y activos
Gestión de curso
Permite administrar el profesor del curso.
Opciones:
Asignar profesor
Cambiar profesor
Mostrar información del profesor
Eliminar curso
Para acceder, primero se debe seleccionar un curso de la lista.
Gestión de estudiantes
Permite administrar los estudiantes de un curso.
Opciones:
Matricular estudiante
Mostrar notas de estudiantes
Eliminar estudiante
Guardar estudiantes en archivo
El sistema permite hasta 30 estudiantes por curso.
Al guardar, se genera un archivo .txt con la información del curso y estudiantes.
5️Gestión de notas
Permite trabajar con las calificaciones de los estudiantes.
Opciones:
Ingresar notas (5 notas por estudiante)
Ordenar estudiantes por promedio
Editar una nota específica
Calcular promedio del curso
Mostrar promedio por evaluación
Reiniciar notas
Las notas válidas están en el rango 0 a 20.
Reporte
Permite visualizar información estadística del curso.
Opciones:
Mostrar estudiante con mayor promedio
Mostrar estudiante con menor promedio
Mostrar resumen del curso
El resumen incluye:
Nombre del curso
Profesor asignado
Total de estudiantes
Aprobados y desaprobados
Promedio general del curso
